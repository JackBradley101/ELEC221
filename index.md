# ELEC 221 Signals and Systems

Welcome to ELEC 221 course notes! This course covers the fundamentals of signals and systems analysis.

## Learning Goals

* Understand continuous-time and discrete-time signals
* Analyze linear time-invariant (LTI) systems
* Apply Fourier analysis techniques to signals and systems
* Use Laplace transforms for system analysis and design
* Implement signal processing algorithms in Python

## About These Notes

These notes are built using [Jupyter Book](https://jupyterbook.org/) and include interactive Python examples.
