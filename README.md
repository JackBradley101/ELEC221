# ELEC 221 Signals and Systems

Course notes for ELEC 221 Signals and Systems, built with Jupyter Book.

**View the site**: [https://jackbradleyis.me/ELEC221/](https://jackbradleyis.me/ELEC221/)

## Building Locally

Install dependencies:
```
pip install -r requirements.txt
```

Build the book:
```
jupyter-book build .
```
